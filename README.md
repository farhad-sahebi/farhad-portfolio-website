# Professional Data Analytics Portfolio Website

I designed and developed this website to showcase my professional work as a Data Analyst, highlighting real-world projects, interactive dashboards, and end-to-end analytics workflows.
It serves as a central portfolio demonstrating my ability to collect, clean, analyze, and visualize data using structured approaches and modern analytical tools.

---

## Main Features
- Portfolio of real-world data projects and dashboards  
- Interactive visualizations and key performance metrics  
- Organized project summaries with clear data workflows  
- Responsive and user-friendly design for all devices  
- Contact section for professional collaboration and communication  

---

## Tools & Technologies Used
- **Excel / Google Sheets** – Data organization and automation  
- **SQL** – Data querying and transformation  
- **R / Python** – Data analysis and reporting  
- **Power BI / Tableau** – Visualization and dashboard creation  
- **HTML, CSS, PHP** – For hosting and structuring this portfolio site
